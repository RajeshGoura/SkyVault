📦 Installation
Clone the repository:

bash
git clone https://github.com/yourusername/skyvault.git
cd skyvault
Install dependencies:
bash
npm install

Set up environment variables:
bash
cp .env.example .env
Then edit .env with your credentials

Start the development server:

bash
npm run dev
